## ğŸ“ Getting Started

### Prerequisites

- Flutter SDK (v3+ recommended)
- Supabase account (for backend)
- Android/iOS emulator or physical device

### Installation

1. **Clone the repo:**
    ```
    git clone https://github.com/YOUR_USERNAME/stukart.git
    cd stukart
    ```

2. **Install dependencies:**
    ```
    flutter pub get
    ```

3. **Set up Supabase:**
    - Register at [supabase.com](https://supabase.com/) and create your project.
    - Set up tables for items, users, and authentication.
    - Add your Supabase credentials as required by the Flutter project.

4. **Run the app:**
    ```
    flutter run
    ```
